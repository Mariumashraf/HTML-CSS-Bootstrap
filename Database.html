Database Fundamentals

the best references and tutorials to study Database

    Books:
  • 1."An Introduction to Database Systems" by Bipin Desai

    Book Review:
    This text is designed to teach the concepts of database systems, from the basic to the relatively sophisticated.
    Each chapter has its own detailed table of contents, a brief introduction, a meaningful summary, a complete list of key terms introduced in that chapter,
    10 to 15 exercises, and a complete bibliography. Fundamental concepts like basic data and file structures, relational data models, network and hierarchical models,
    query processes, languages, data recovery, database security and integrity, distributed databases, concurrency control, distributed locking, and the two-phase commit process.
    This book can be referred to in the senior year of an undergraduate program or the first year of graduate study.

  • 2."Fundamentals of Database Systems" by R Elmasri and S Navathe

    Book Review:
    The book provides clear cut information and explanation about the various concepts of database management systems
    from both the angles of theory and design. The book also introduces modern database technologies thereby providing the explanation about
    the database systems. This book is very useful for computer science professionals. The book focuses on math models, design issues,
    relational algebra and relational calculus. The book contains various problems which provides the students an opportunity to deal with the basics of design and implementation.
    The book covers advanced topics like SQL, data mining and security, XML and semi-structured data.
